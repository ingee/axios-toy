# How api-server is created
```
$ npm install -g express-generator
$ create api-server
$ cd api-server
$ npm install
$ npm install --save-dev nodemon
# Add 'devstart' and 'serverstart' scripts in package.json
```

# Usage
```
### At api-server directory,
$ npm run serverstart

### At another terminal
$ curl localhost:3000/users
```
