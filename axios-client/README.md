# How axios-client is created
```
$ npm install -g @vue/cli
$ vue create axios-client
  ### select vue2, router, vuex, lint-standart
$ cd axios-client
$ npm install
$ vue add vuetify
  ### select 'default'
$ npm install --save axios
```
